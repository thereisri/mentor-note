# m-note

> 팀 업무 노트 & 프로젝트 관리 웹앱

작은 회사에서 일하다 보면 업무 메모, 수정 내역, 테스트 결과가 여러 군데 흩어져서
프로젝트 흐름을 파악하기 어렵고, 사람마다 작업 방식이 달라 품질이 들쭉날쭉해집니다.
**m-note**는 이 문제를 해결하기 위해 만든 팀 협업용 업무 노트 앱입니다.

<br/>

## 📌 기획 배경

- 업무 내용·수정 내역·테스트 결과가 카카오톡, 엑셀, 메모장에 분산
- 정해진 프로세스가 없어 사람마다 결과물 품질 편차 발생
- 프로젝트를 노트로 공유하면 **표준 프로세스를 자연스럽게 정립**할 수 있지 않을까?

<br/>

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| **플로우 노트** | 프로젝트를 구성하는 노트들을 화살표로 연결해 전체 흐름을 한눈에 파악 |
| **디테일 노트** | 개별 노트에 설명·필수사항·체크리스트·추가 요소를 구조화해서 기록 |
| **상태 관리** | 완료 / 진행 / 대기 상태를 리본 배지로 시각화 |


<br/>
